### A bit about me
I first picked up interest in programming in 2015. Since then I've gone from Roll-A-Ball Unity tutorials, to making Discord bots, websites, rendering systems with a variety of APIs, and even a functional game engine with multiple games built and maintained on top of it. I attend Full Sail University for undergrad, and am currently attending SMU Guildhall for to earn my Masters.<br/>
When it comes to programming, I've found great joy in solving pain points for others, usually in the form of tooling. If there are pipeline or workflow pains, you can bet you'll find me poking about trying to learn from those it affects how best I may be able to help smooth things out. I take pride and purpose in seeing the things I create provide value directly to those around me.

### Featured projects
- **GuildhallProjects**: A collection of projects from my time at Guildhall. Includes the custom C++ game engine created and maintained during my time at Guildhall.
- **FireheadEngine**: A custom Vulkan renderer supporting loading models, and instanced rendering. Uses CMake, glfw, and bootstrapping to make project setup as portable as possible.
- **Unreal-Project-Tools**: A small variety of automation scripts for use with Unreal Engine C++ projects using perforce. The highlight script is the rebuild script which makes running a full, clean build of the editor for an Unreal project simple enough to hand off to none technical users to use.
